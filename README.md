# 🚗 鸿蒙智能座舱系统

## 项目简介

本项目是**2025年广东省大学生计算机设计大赛**的参赛作品（作品编号：2025038208），基于 **OpenHarmony** 系统开发的智能车机仪表盘应用。采用 **ArkTS** 语法和鸿蒙设计语言，打造简洁美观、操作便捷的现代化车机界面，为驾驶者提供沉浸式的智能座舱体验。

## ✨ 核心功能

### 📱 主页面
- **卡片式布局**：导航、天气、音乐、仪表盘、主题等模块采用卡片样式
- **车机状态模块**：实时监测并显示车辆关键参数
- **环境感知**：集成天气信息展示，支持雨雪特效
- **车辆控制**：提供驾驶模式选择、鸣笛控制等功能

### 🎨 仪表盘模块
- **速度表盘**（左侧）：带平滑动画效果，实时显示车速
- **转速表盘**（右侧）：流畅动画效果，显示发动机转速
- **转向指示灯**：左右转向灯动态闪烁效果
- **档位显示**：P/R/N/D/S 档位实时切换
- **空调控制**：温度调节、风量控制

### 🧭 车辆状态页面
- **方向盘角度反馈**：硬件同步，实时显示方向盘转角
- **车辆自检**：动画展示自检进度，包括车胎、水压、机油压力等
- **驾驶模式切换**：支持标准、经济、运动、雪地模式

### 🌤️ 天气模块
- **智能定位**：通过 WLAN/网络定位获取地理位置
- **逆地理编码**：经纬度转城市/区县信息
- **实时天气**：温度、湿度、天气状况展示
- **天气特效**：根据天气情况展示雨雪动画

### 🎵 音乐模块
- **完整播放控制**：播放、暂停、上一曲、下一曲
- **播放列表**：懒加载展示歌曲列表
- **专辑封面**：滑动切换专辑图片
- **评论展示**：查看用户评论
- **状态持久化**：保存播放状态和用户偏好

### 🎨 主题与个性化
- **多种主题**：经典黑红、科技蓝、极简白、清新绿
- **自动主题切换**：白天（6-18点）深色主题，夜晚浅色主题
- **亮度调节**：滑块控制屏幕亮度，持久化存储
- **偏好设置**：转向灯提示音、驾驶模式音效、语音反馈等

### 🚀 启动过渡页
- **渐进式动画**：车辆滑入 → 标题淡入 → 进度条填充 → 提示显示
- **数据预加载**：动画期间并行获取地理位置和天气数据
- **流畅过渡**：缓动曲线提升动画自然度
