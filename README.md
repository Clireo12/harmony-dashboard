# 🚗 OpenHarmony 智能车机仪表盘

## 项目简介

本项目是**广东省计算机设计大赛**的参赛作品，基于 **OpenHarmony** 系统开发的智能车机仪表盘应用。采用 **ArkTS** 语法和鸿蒙设计语言，打造简洁美观、操作便捷的现代化车机界面，为驾驶者提供沉浸式的智能座舱体验。

## ✨ 核心功能

### 📱 主界面
- **卡片式布局**：导航、天气、音乐、仪表盘等模块采用卡片样式
- **实时车机状态**：监测并显示车辆关键参数
- **响应式设计**：完美适配不同尺寸的车机屏幕

### 🎨 仪表盘模块
- **速度表盘**（左侧）：带平滑动画效果，实时显示车速
- **转速表盘**（右侧）：流畅动画效果，显示发动机转速
- **转向指示灯**：左右转向灯动态效果
- **档位显示**：P/R/N/D 档位实时切换
- **空调控制**：温度调节、风量控制

### 🧭 导航模块
- 实时地图导航
- 路线规划
- 语音播报

### 🌤️ 天气模块
- **智能定位**：自动获取当前地理位置
- **实时天气**：温度、湿度、天气状况

### 🎵 音乐模块
- 音乐播放控制
- 歌曲信息展示
- 点击跳转至完整音乐播放器

### 🚗 车辆控制
- **方向盘角度反馈**：硬件同步，实时显示方向盘转角
- **车辆自检**：点击中央车辆图标进入自检页面
- **故障诊断**：车辆健康状态检测

## 🛠 技术栈

| 技术 | 说明 |
|------|------|
| **开发平台** | OpenHarmony|
| **开发语言** | ArkTS (TypeScript 超集) |
| **UI框架** | ArkUI |
| **设计语言** | 鸿蒙设计规范 |
| **状态管理** | @State, @Link, @Observed |
| **动画系统** | 属性动画、显式动画 |
| **定位服务** | 地理位置 API |
| **硬件交互** | 传感器 API |
